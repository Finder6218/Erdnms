# 专家随机抽号管理系统

《专家随机抽号管理系统》是专门为医院随机抽取专家而开发的一款软件。软件操作简便、方便上手,功能简洁明了，是管理者的好帮手！ 系统包含以下功能： 业务管理：抽取专家、按项目名称查询、按专家姓名查询； 信息维护：专家信息、单位信息。
